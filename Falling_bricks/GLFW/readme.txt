
--> The Player has 3 lives to play the game. THe game ends after 3 lives finish.
--> SOUND is played when laser gets out of the canon.
--> SCORE Is Displayed on top right corner using 7 Segment Display


--> CONTROLS:
* CANON -> TILT-UP = A
		-> TILT-DOWN = D
		-> MOVE-UP = S (or Click and Drag)
		-> MOVE->DOWN = F (or Click and Drag)

* SPEED -> UP = M
		-> DOWN = N
(within the permissible limits)

* BASKETS -> BASKET1 LEFT = CNTRL + LEFT (or Click and Drag)
		  -> BASKET1 RIGHT = CNTRL + RIGHT (or Click and Drag)
		  -> BASKET2 LEFT =ALT + LEFT (or Click and Drag)
		  -> BASKET2 RIGHT = ALT + RIGHT (or Click and Drag)

* SHOOT LASER -> SPACE
* ZOOM -> UP/DOWN ARROW KEYS (or Scroll Up/Down)
* PANNING -> LEFT/RIGHT ARROW KEYS (Or RIght Click and Drag within the permissible limits of screen)

--> LEVEL UP after a score of multiple of 100 (speed of bricks increases)

-->Score: (Always Positive)
* SHOOT:   -> Black = Life-1
	       -> Any other = +20
* COLLECT: -> Black = Life-1
	       -> Red = By Red Basket +10
	       		  = By Blue Basket -1
	       -> Blue = By Blue Basket +10
	       		  = By Red Basket -1
